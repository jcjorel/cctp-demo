Introduction

Cette bibliothèque contient tous les composants UI utilisés dans les mockups du Système de Réservation de Ressources. Elle sert de référence pour maintenir une cohérence visuelle et fonctionnelle à travers toutes les interfaces. Les composants sont inspirés du Material Design et préparés pour une migration future vers React avec Material-UI.

Notifications

Les notifications informent les utilisateurs d'événements importants, de confirmations ou d'erreurs. Leur couleur indique visuellement la nature du message.

✓

Succès

La réservation a été créée avec succès.

i

Information

Votre demande est en attente de validation.

!

Avertissement

Cette ressource sera bientôt indisponible pour maintenance.

⚠

Erreur

La réservation n'a pas pu être enregistrée.
